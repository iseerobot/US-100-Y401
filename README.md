# US-100-Y401
Example code for US-100 (Y401)

![alt tag](http://www.iseerobot.com/img/foto_produk/15sensor%20US-100.jpg)

## Wiring ##
| PIN US-100    | PIN Arduino  UNO |
| ------------- | ---------------- |
| VCC           | 5V               |
| GND           | GND              |
| TRIGGER       | Digital 6        |
| ECHO          | Digital 7        |

This code is tested by iseerobot.com
