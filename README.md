# US-100-Y401
Contoh program untuk US-100 (Y401)

## Wiring ##
| PIN US-100    | PIN Arduino   |
| ------------- | ------------- |
| VCC           | 5V            |
| GND           | GND           |
| TRIGGER       | Digital 6     |
| ECHO          | Digital 7     |
